# 目标
为编程初学者打造入门学习项目，使用各种主流编程语言来实现。让想学编程的，一个都不落下。
![](https://img-blog.csdnimg.cn/52e58180eab0496ebff3e830f85be5d8.png)

上述基本涵盖了当前编程开发所有主流语言。  
左侧为前端版本：安卓、iOS、鸿蒙、Flutter、Vue、uni-app、微信小程序。  
右侧为服务器端版本：Java、Python、Go、Php、NodeJS
